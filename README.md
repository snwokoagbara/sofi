# Special Ones Foundation (SOFI) Website

Modern website for the Special Ones Foundation, a nonprofit organization supporting children with developmental differences and providing meals to underserved children.

## Pages

- **Home** (`index.html`) - Main landing page with mission and impact counters
- **Our Work** (`our-work.html`) - Information about education and meal programs
- **Support Us** (`support-us.html`) - Donation information and links
- **About Us** (`about-us.html`) - Organization background and team
- **Contact** (`contact.html`) - Contact form and information

## Features

- Responsive design
- Impact counters (Kids Fed, Meals Served)
- Financial transparency link
- Modern, accessible UI

## Deployment

This is a static website that can be deployed to:
- GitHub Pages
- Netlify
- Vercel
- Any static hosting service

## Local Development

To preview locally, run:
```bash
python3 -m http.server 8000
```

Then open http://localhost:8000 in your browser.
